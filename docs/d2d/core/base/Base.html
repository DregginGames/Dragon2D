    Class Base

    Base is the base interaface for all engine classes. It has the methods that are basically needed by every part of the system.

    Inherits from

    • Object (base class)

    this Base ctor

    Properties

    Name Type Description
    acceptsEvents [get] bool if this object accepts events
    children [get] Base[long] gets the child objects of this object
    curticks [get] long returns the ticks since engine creation.
    curtime [get] long returns the time since engine/root creation (clipped to ticks)
    curtimeS [get] double returns the time since engine/root creation in seconds (clipped to ticks)
    deleted [get] bool returns if the object will be deleted before the next tick
    id [get] ulong fixed obj id of this object
    objCurtime [get] long return the time since object creation (clipped to ticks)
    objCurtimeS [get] double return the time since object creation, in seconds (clipped to ticks)
    parent [get] Base gets the parent object of this object
    parent [set] Base sets the parent object of this object
    paused [get] bool pauses/unpauses the object
    root [get] Base the "root" is the root of the element tree wich this object is a member
    ticktime [get] long returns the length of the last tick
    ticktimeS [get] double returns the length of the last tick in seconds

    addChild add a child to this object. Resets the parent of the object to add.

    getService Gets a service by its name. Syntax is getService!ServiceClass(name)

    postUpdate postUpdate is called after update is called

    propagateUpdate Propagates an update thru the object hirarchy
    removeChild removes a child from this object
    removeSaveRestoreHandler Removes a SaveRestore handler
    render Render is called every frame
    restoreSave Restores stuff
    restoreSaveValue Gets a value from the savable data storage
    save Saves stuff
    setDeleted marks an object for deletion. It immediatly removes the service (if existant) and also marks all children for deletion! Deletion means removal from object hirarchy and services. However if the object is stored somewhere else it also has to be removed manually there (should never ever happen).
    storeSaveValue Sets a value into the saveable data storage
    update Update is called every tick (1/30 of a second).

    onChildRemoved Called if a child is removed from this object

    propagate Propagates event actions
