    Class ColoredQuad

    A simple single colored

    Inherits from

    • Quad (base class)

    Properties

    Name Type Description
    color [get] gl3n.linalg.Vector!(float,4) The color of the quad
    detailLevel [get, set] int The detail Level. The higher this level is, the higher the level of a view needs to be to see this object
    ignoreView [get, set] bool Renderables can be set to ignore the effects of views (other than viewport), making, for them, -1..1 direkt mapping of screen space. This can be usefull for the user interface, debug text, things that should annoy people, generally everything you dont want to use in world but in screen coordinates
    pos [get, set] gl3n.linalg.Vector!(float,2) The position of this renderable in world-coordinates
    program [get] GlslProgram The program of this quad
    rotation [get, set] float The rotation of the renderable (around z axis)
    scale [get, set] gl3n.linalg.Vector!(float,3) The scale of this renderable in world-coordinates

    Methods

    Name Description
    render Performs an actual render on screen.
    _setupVAO Sets the mode with wich the classes VAO is managed. If set once, it cant be changed - means that there is a need to take care how the interaction with parten-classes is handled
    _standardModelToWorld Returns the model to world matrix for the combination of position, scale and rotation
    _vboInit Called by setupVAO if the mode fór VBO is objectScope
    _vboInitClassScope Called by setupVAO if the mode for VBO is classScope
