    Class RawTexturedQuad

    A simple textured quad

    Inherits from

    • Quad (base class)

    Fields

    Name Type Description
    _drawOffset gl3n.linalg.Vector!(int,2) Offset for the draw call. For RawTexture defaults to 0,6 - the stuff needed for a square

    Properties

    Name Type Description
    texture [get] GPUTexture The texture of this quad
    detailLevel [get] int The detail Level. The higher this level is, the higher the level of a view needs to be to see this object
    pos [get] gl3n.linalg.Vector!(float,2) The position of this quad. /-- totally not stolen from entity.d
    program [get] GLSLProgram The program of this quad

    Methods

    Name Description
    getUVOffset The uv-offset of this quad
    render Performs an actual render on screen.
    _vboInitClassScope Fills the UV- and Vertex buffers with thier Data.
    _setupVAO Sets the mode with wich the classes VAO is managed. If set once, it cant be changed - means that there is a need to take care how the interaction with parten-classes is handled
    _vboInit Called by setupVAO if the mode fór VBO is objectScope
