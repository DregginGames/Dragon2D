    Class Renderable

    Renderable is abstract base for all types of /renderables/. A Renderable is an object that actually is renderd on screen, like a circle, a textured quad, a tile map, .... Its one of the few structures that life outside the Base-class hirarchy

    An importand concept of Renderables is the detail level. The higher this level is, the higher the level of the View needs to be to be able to see this object. Here is a reccomendation for the detail levels:

    00..19: Application-level - always-visible things: Debug information, Alerts, ....

    20..39: Map-level - map-baisc stuff like terrain

    40..59: Map-detail-level - advaned stuff like Buildings, details, flowers, ...

    60..79: Actor-level - everything that actually does things like NPCs, players, ...

    80+: UI-Level - the user interface. Above that... whatever
