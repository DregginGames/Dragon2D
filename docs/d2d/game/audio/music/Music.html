    Class Music

    A music

    Inherits from

    • Base (base class)

    Constructors

    Name Description
    this Creates a music from the given resource name

    Properties

    Name Type Description
    musicPaused [get] bool If the music is paused. Since paused is reserved by base, this name is a bit silly
    playing [get] bool If the music ist the active one, has been started and is not paused
    started [get] bool If the music is the active one and started, regardless if it later has been paused or not
    volume [get, set] int The volume with wich the music should be played, from 0-128. Values above or below will map to the nearest valid value
    acceptsEvents [get] bool if this object accepts events
    children [get] Base[long] gets the child objects of this object
    deleted [get] bool returns if the object will be deleted before the next tick
    id [get] ulong fixed obj id of this object
    parent [get] Base gets the parent object of this object
    parent [set] Base sets the parent object of this object
    paused [get] bool pauses/unpauses the object
    root [get] Base the "root" is the root of the element tree wich this object is a member

    Methods

    Name Description
    pause This will pause a paying Music. If the music is already paused or not playing, this will have no effect
    pauseActive Pauses the currently active music
    play Will start playing the music from the beginning.
    resume This will resume a paused Music. If the music is not paused or not playing in the first place this will have no effect
    resumeActive Resumes the currently active music
    stop Stops a started music. If the music is not started (so can be paused but play should have been called at least once), nothing will happen
    stopActive Stops the currently active music
    addChild add a child to this object. Resets the parent of the object to add.
    fireEvent Fires an event, moves it up in the object hirarchy and propagetes it through
    getService Gets a service by its name. Syntax is getService!ServiceClass(name)
    postRender postRender is called after all children and the object were renderd.
    postUpdate postUpdate is called after update is called
    preTickDelete PreTickDelte removes all objects that are makred for deletion from the hirachy. If an objects had children, these are deleted, too.
    propagateEvent propagate an event throgh the object hirarchy
    propagateRender Propagates the rendering through the object hirarchy
    propagateUpdate Propagates an update thru the object hirarchy
    removeChild removes a child from this object
    render Render is called every frame
    setDeleted marks an object for deletion. It immediatly removes the service (if existant) and also marks all children for deletion! Deletion means removal from object hirarchy and services. However if the object is stored somewhere else it also has to be removed manually there (should never ever happen).
    update Update is called every tick (1/30 of a second).
    enableEventHandling Enables event reciving for this object. Should be called in the constructor of classes that want it.
    pollEvents returns the pending events and cleans
    propagate Propagates event actions
    registerAsService Registers a class as a service that can be accessed by its name. Should be called by classes that want to be deleted
    removeService Removes a service
