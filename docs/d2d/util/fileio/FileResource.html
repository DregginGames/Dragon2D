    Class FileResource

    this class manages and reprresents all files that are read/written to the system

    Inherits from

    • Object (base class)

    Properties

    Name Type Description
    file [get] string gets the file that this resource belongs to
    invalid [get] bool gets if the file resource is invalid
    modified [get] bool get if the resource was modified
    name [get] string gets the name of the file resource
    read [get] bool gets if the resource has been read

    Methods

    Name Description
    createGameResource creates a new (read as: never exsisted) resource in game path
    flush flushes a file to the system
    flushAll flushes all know resources. might be slow...
    flushAndClean flushes a file to the system and them removes cached data
    getData gets the date of i file. if not yet done, reads them from the file system
    getFileResource returns a FileResource for the given resource stirng resource strings have the format resourcetype.name or resourcetype.subtpe.name or foo.bar.x.y.z - importand is that this is transformed into a path without file extension (foo/bar/x/y/z.*). THAT MEANS THAT FILENAMES WITHOUT EXTENSION MUST BE UNIQUE IN A DIRECTORY! THE FUNCTION WILL IGNORE ALL BUT THE FIRST ENTRY FOUND!
    nameToPath returns string[2], [0] is the basename, [1] is the path
    setData sets the data of this file. Since data are overwritten in the end logically causes read to be set to true. Flush is needed to store them to the system!
